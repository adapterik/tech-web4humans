# web4humans Tech Info Site

## Overview

This repo contains an implementation of a small web app, used to run a site under the "web4humans" umbrella, discussing various issue of web and internet technology.

This project was originally developed on macOS - but I have since switched over to Linux full-time for everything. I am in the process of both switching over to  a Linux workflow and reworking the project to be static and generated rather than a live Ruby web server with content generated on the fly. The reason for these changes will be explained in an article on the site itself!
